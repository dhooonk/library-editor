## 4. 트리뷰 탐색

- **📁 LIB: 이름** — `.LIB` 블록 (클릭 시 해당 블록 정보 표시)
- **🔧 PARAMS** — `.PARAM` 변수 목록 편집
- **⚙️ MODEL명 [타입]** — 특정 모델 선택 → 우측에 파라미터 목록 표시
- **🔧 전역 PARAMS** — 파일 전체에 적용되는 전역 `.PARAM` 변수

> 항목을 클릭하면 우측 편집 패널이 해당 내용으로 전환됩니다.

---

## 5. 파라미터 편집

### 값 수정
1. 우측 테이블에서 수정할 셀을 **더블클릭**
2. 값 입력 후 **Enter** 또는 **Tab** → 저장  
   **Esc** → 취소

> 파라미터 **이름**(왼쪽 열)과 **값**(오른쪽 열) 모두 더블클릭으로 수정 가능합니다.

### 파라미터 추가
1. 좌측 트리에서 **MODEL** 또는 **PARAMS** 선택
2. **`＋ 파라미터 추가`** 버튼 클릭
3. 이름과 값 입력 후 **확인**

### 파라미터 삭제
1. 삭제할 파라미터 행을 **클릭하여 선택**
2. **`－ 삭제`** 버튼 클릭 → 확인 창에서 **예**

## 6. 변수(PARAM) 사용법

파라미터 값에 고정 숫자 대신 **변수**를 사용하면, 나중에 변수 하나만 바꿔도 여러 모델에 일괄 반영됩니다.

### 변수 할당 방법

1. 변수로 연결할 **파라미터 행 선택**
2. **`🔤 변수 → 값 설정`** 버튼 클릭
3. 대화창에서:
   - **변수 이름** 입력 (예: `tox_val`)  
     기존 변수가 있으면 드롭다운에서 선택 가능
   - **변수 값** 입력 (예: `1.2e-8`)
   - **수식** 확인/수정 (자동 생성됨, 예: `{tox_val}`)
4. **적용** 클릭

→ 결과: 파라미터 값이 `{tox_val}` 로 바뀌고, 전역 `.PARAM tox_val=1.2e-8` 가 자동 추가됩니다.

### 수식 예시

| 수식 표현 | 의미 |
|----------|------|
| `{tox_val}` | 변수 값 그대로 사용 |
| `{tox_val * 1.1}` | 변수 값 × 1.1 |
| `{vth_offset + 0.40}` | 변수 값 + 0.40 |
| `{K1 / 2}` | 변수 값 ÷ 2 |

> 수식은 Smart Spice가 시뮬레이션 시 자동으로 계산합니다.

### PARAMS 노드에서 직접 변수 추가/수정
1. 트리에서 **🔧 PARAMS** 선택
2. 우측 테이블에 변수 목록 표시됨
3. 더블클릭으로 변수명/값 수정, `＋` 버튼으로 신규 추가, `－` 로 삭제

## 7. 색상 코딩

| 색상 | 의미 |
|------|------|
| 🟢 **초록** | 숫자 값 (`0.45`, `1.2e-8`) |
| 🟡 **노랑** | 변수 단순 참조 (`{tox_global}`) |
| 🟣 **보라** | 연산 수식 (`{vth_offset+0.40}`, `{k1 * 2}`) |

## 8. 저장된 파일 형식 (Smart Spice 문법)

편집 후 저장하면 아래와 같은 형식으로 출력됩니다:

```spice
.PARAM tox_global=1.2e-8

.LIB NMOS_LIB
.PARAM vth_offset=0.05
.MODEL NMOS_V1 NMOS
+ LEVEL=3 VTH0=0.45 TOX={tox_global} XJ=1.5e-7
+ K1=0.559 K2=-0.04
.ENDL NMOS_LIB
```

- 긴 줄은 `+` continuation line으로 자동 줄바꿈 (80자 기준)
- `.PARAM` 변수는 LIB 블록 앞 또는 전역으로 선언
