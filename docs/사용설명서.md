## 3. 파일 열기 / 저장

### 파일 열기

1. 상단 툴바에서 **📂 파일 열기** 버튼을 클릭합니다.
2. 파일 선택 창에서 편집할 `.lib` 파일을 선택합니다.
3. 파일이 열리면 좌측 트리뷰에 구조가 나타납니다.

### 저장

| 버튼 | 동작 |
|------|------|
| **💾 저장** | 열려있는 파일을 **원본 경로에 덮어씁니다** |
| **💾 다른 이름으로 저장** | 새 파일명과 경로를 지정하여 **별도 파일로 저장**합니다 |

> ⚠️ **저장 전 주의**: 편집 내용은 자동 저장되지 않습니다. 중요한 변경 후에는 저장 버튼을 반드시 누르세요.

## 4. 좌측 트리뷰 탐색

좌측 트리뷰는 `.lib` 파일의 계층 구조를 시각적으로 보여줍니다.

| 아이콘 | 의미 |
|--------|------|
| **📁 LIB: 이름** | `.LIB … .ENDL` 블록 (클릭하면 블록 요약 정보 표시) |
| **🔧 PARAMS** | 해당 영역의 `.PARAM` 변수 목록 |
| **📄 설정 (.directives)** | `.temp`, `.global` 등 기타 설정 지시어 목록 |
| **⚙️ MODEL명 [타입]** | 개별 `.MODEL` 항목 (클릭하면 파라미터 목록 표시) |
| **🔧 전역 PARAMS** | `.LIB` 블록 바깥의 전역 `.PARAM` 변수 |
| **📄 전역 설정** | `.LIB` 블록 바깥의 전역 지시어 |

**항목 클릭** → 우측 편집 패널에 해당 내용이 표시됩니다.

### LIB/MODEL 이름 변경

1. 변경하려는 **📁 LIB** 또는 **⚙️ MODEL** 항목을 **더블클릭**합니다.
2. 팝업 창에서 새 이름(또는 타입)을 입력하고 **확인**을 클릭합니다.

---

## 5. 파라미터 편집

### 파라미터 값/이름 수정 (인라인 편집)

1. 좌측 트리에서 수정할 **⚙️ MODEL** 항목을 클릭합니다.
2. 우측 테이블에 파라미터 목록이 표시됩니다.
3. 수정할 셀(이름 또는 값)을 **더블클릭**합니다.
4. 셀 위에 입력창이 나타납니다 — 새 값을 입력합니다.
5. **Enter** 또는 **Tab** → 저장 / **Esc** → 취소

> 💡 **수식 입력 방법**: 변수를 참조하거나 수식을 쓸 때는 중괄호를 사용합니다.
> - 변수 참조: `{tox_global}`
> - 산술 수식: `{vth_offset * 1.1}`, `{k1 + 0.05}`

## 6. 파라미터 추가 / 삭제

### 파라미터 추가

1. 좌측 트리에서 추가할 위치(MODEL 또는 PARAMS)를 선택합니다.
2. 하단 **＋ 파라미터 추가** 버튼을 클릭합니다.
3. 팝업 창에서 **이름**과 **값**을 입력합니다.
4. **확인**을 클릭하면 테이블에 새 행이 추가됩니다.

### 파라미터 삭제

1. 우측 테이블에서 삭제할 파라미터 행을 클릭(선택)합니다.
2. 하단 **－ 삭제** 버튼을 클릭합니다.
3. 삭제 확인 메시지에서 **예**를 선택합니다.

## 7. 일괄 수정

여러 모델에 걸쳐 동일한 파라미터를 한 번에 변경할 수 있습니다.

1. 하단 **📋 일괄 수정** 버튼을 클릭합니다.
2. 팝업 창에서 설정합니다:

   | 항목 | 설명 |
   |------|------|
   | **대상 파라미터** | 변경할 파라미터명을 드롭다운에서 선택 |
   | **새 파라미터 값** | 새로 적용할 값 입력 |
   | **적용 범위** | `모든 LIB` — 파일 전체 / `현재 보고있는 LIB` — 현재 선택 블록만 |

3. **적용** 버튼을 클릭하면 해당 이름을 가진 모든 모델의 값이 일괄 교체됩니다.
4. 완료 후 변경된 모델 수가 알림 메시지로 표시됩니다.

## 8. 파라미터 중심 뷰

특정 파라미터에 대해 모든 LIB·모델의 값을 한 화면에서 비교할 수 있습니다.

1. 좌측 트리 하단의 **🔄 파라미터 중심 뷰 열기** 버튼을 클릭합니다.
2. 새 창이 열립니다:
   - **좌측**: 파일 전체 파라미터명 목록
   - **우측**: 선택한 파라미터의 LIB명 / MODEL명 / 값 비교 테이블
3. 좌측에서 파라미터명을 클릭하면 우측에 모델별 값이 나타납니다.

## 9. 변수(PARAM) 활용법

`.PARAM` 변수를 사용하면 변수 하나만 바꿔도 여러 모델에 자동 반영됩니다.

### 변수 만들기

1. 좌측 트리에서 **🔧 전역 PARAMS** 또는 **🔧 PARAMS** 노드를 클릭합니다.
2. 하단 **＋ 파라미터 추가** 버튼을 클릭합니다.
3. 이름(`tox_val`)과 값(`1.2e-8`)을 입력하고 **확인**을 클릭합니다.

### 변수 참조하기

모델 파라미터 값 입력 시 중괄호 `{ }` 안에 변수명을 입력합니다.

| 입력 예시 | 의미 |
|----------|------|
| `{tox_val}` | tox_val 변수 값 그대로 사용 |
| `{tox_val * 1.1}` | tox_val × 1.1 |
| `{vth_offset + 0.40}` | vth_offset + 0.40 |
| `{K1 / 2}` | K1 ÷ 2 |

> 수식은 Smart Spice 시뮬레이터가 실행 시 자동으로 계산합니다.

## 10. Excel 내보내기

파일 전체 데이터를 Excel(.xlsx)로 출력합니다.

1. 툴바의 **📊 Excel 내보내기** 버튼을 클릭합니다.
2. 저장 위치와 파일명을 지정합니다.  
   (기본 파일명: 원본 파일명 + `_export.xlsx`)
3. **저장**을 클릭합니다.

생성되는 Excel 시트 구성:

| 시트 이름 | 구성 |
|----------|------|
| **Matrix View** | 가로=모델명, 세로=파라미터명 — 값 비교 행렬 |
| **List View** | Library / Model / Type / Parameter / Value 1차원 목록 |

우측 파라미터 테이블의 값 색상은 값의 종류를 나타냅니다.

| 색상 | 의미 | 예시 |
|------|------|------|
| 🟢 **초록** | 숫자 리터럴 | `0.45`, `1.2e-8`, `3` |

## 15. 자주 묻는 질문 (FAQ)

**Q. 파일을 열었는데 트리뷰에 아무것도 안 보입니다.**  
A. `.lib` 파일이 `.LIB … .ENDL` 구조를 갖추고 있는지 확인하세요.
   파일 최상단에 `*` 주석만 있는 경우 전역 PARAMS/설정만 표시될 수 있습니다.

**Q. 저장 후 파일을 열어보면 줄바꿈이 달라졌습니다.**  
A. 80자 기준으로 `+` continuation line이 자동 재배치됩니다.
   내용(값)은 동일하게 보존됩니다. Smart Spice 시뮬레이션에는 영향이 없습니다.

**Q. 원본 주석(`*`)이 사라졌습니다.**  
A. `*` 주석은 바로 다음 `.LIB` 블록 또는 `.MODEL` 선언에 귀속됩니다.
   파일 최상단 주석은 `.LIB` 블록이 없을 때만 보존됩니다.

**Q. Excel 내보내기가 실패합니다.**  
A. `openpyxl` 라이브러리가 설치되지 않은 경우입니다.
   `pip install openpyxl` 명령어를 실행한 후 다시 시도해 주세요.

**Q. 테마를 전환하면 파일 데이터가 초기화됩니다.**  
A. 테마 전환 시 UI를 완전히 재구성하지만, 메모리에 로드된 파일 데이터는 유지됩니다.
   파일을 다시 불러올 필요가 없습니다.
